Defined in kola\html\util\Event.js:178

监听一个事件

Parameters:

  • element KolaElement

    要绑定事件的元素

  • name String

    事件名称

  • callback Function

    事件的处理函数

  • [options] Object optional

    配置参数

    • [scope] Object optional

      指定处理函数的this，如果没有，则默认为element

    • [data] ANY optional

      绑定事件时附带的参数，事件处理时会附加在回调函数的参数后面

    • [delegate] Stirng optional

      代理事件，如果设置，只有符合该选择器的子元素才会触发事件，并且currentTarget指向被代理的元素

  • [hook] Object optional

    事件钩子，用于定制特殊事件

    • [setup] Function optional

      事件绑定前拦截

    • [teardown] Function optional

      事件解绑前拦截

    • [domEvtName] String optional

      对应的dom事件名称

    • [domEvtName] String optional

      对应的dom事件名称
